## 📖 Overview

Karen Talks is an AI-powered debate platform that enables users to engage in dynamic discussions with AI-generated debate partners. Instead of providing generic chatbot responses, the application simulates structured debates with different personalities, perspectives, and communication styles.

Whether you're preparing for interviews, practicing public speaking, improving critical thinking, or simply exploring different viewpoints, Karen Talks offers an engaging conversational experience.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Karen_Talks.git
```

Move inside the project

```bash
cd Karen_Talks
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
streamlit run Karen.py
```

## 💡 How It Works

1. Launch the application.
2. Choose your preferred debate style.
3. Select an AI debate personality.
4. Enter your topic or opinion.
5. Gemini AI generates intelligent debate responses.
6. Continue the discussion naturally.
